Exercice : "FizzBuzz".

Exercice page 37, Program Structure Lesson.

Write a program that uses console.log to print all the numbers from 1 to the defined FizzBuzz max value, with two exceptions. For numbers divisble by 3, print "Fizz" instead of the number, and for numbers divisible by 5 and not 3), print "Buzz" instead.

When you have that working, modify you program to print "FizzBuzz" for numbers that are divisible by both 3 and 5 (and still print "Fizz" or "Buzz" for numbers divisible by ony one of those).
